Object Literals

  1. A self-contained set of related values and functions
  2. Incased in {} and a distinguishing feature of Java Script
  3. Each property is a key-value separated by commas.
  4. Creating an Object
    1. const objectname = { item: 'itemname', item: 'itemname', multipleitems: ['item','item','item'], method () { return 'something';}};
  1. Accessing Properties
    1. itemname.item
    2. itemname['item']
  1. Computed Properties
    1. ES6 can concatenate using ['item' + 'item']
    2. Can return a True or False based on the value of a variable: item: itemname ? false : true
    3. Symbol data type: const name = Symbol ('name')
  1. Calling method
    1. itemname.methodname()
  1. Checking if Properties or Methods Exist
    1. 'property' in item;
  1. Finding all properties of an Object
    1. Use a "for in" loop
    2. Use the Object.keys() method to find the keys in an object
    3. Use the Object.values() method to find the values of an object
    4. Use the Object.entries() method to return keys and values
  1. Adding Properties
    1. value.newproperty = 'item';
  1. Changing Properties
    1. Use item['property'] = 'newvalue';
  1. Removing Properties
    1. Use the delete operator
  1. This
    1. Refers to the object is within
  1. Namespacing
    1. Errors occur when a variable or function share the same name
  1. JSON
    1. Lightweight data storage
  1. The Math Object
    1. Built-in object used for math operations

 

Document Object Model

  1. Getting Elements
    1. getElementById() method
    2. getElementByTagName() method
    3. getElementByClassName() method
    4. querySelector() method allows to use CSS notation to find the first element in a document that matches
    5. querySelectorAll() method finds all the elements that match
  1. Navigating DOM
    1. Use childNodes to list all the children of a certain node
    2. Use firstChild to list first child and lastChild to list last child of the node
    3. Use parentNode to find the parent node of a certain node
    4. Use nextSibling to return the next adjacent node
    5. Use perviousSibling to return the pervious node
  1. Getting and setting attributes
    1. Use getAttribute() method to get attributes
    2. Use setAttribute() method to set attributes
  1. Creating Dynamic Markup
    1. Use createElement() method to create an element on a webpage
    2. Use createTextNode() method to create and put text into a node
    3. Use insertBefore() method to place a new element prior to a specified element
    4. Use insertAfter() method to place a new element following a specified element
    5. Use removeChild() to remove a specified node
    6. Use replaceChild() to replace a specified node
    7. Use innerHTML to place a chunk of HTML inside and element
  1. Updating CSS
    1. Use the style property to change CSS styles in a document
