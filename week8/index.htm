Transforms and Transitions

 

Transform Property

  1. Let's you translate, rotate, scale, and/or skew any element on the page.
  2. Translation
    1. Allows you to move elements left, right, up or down without effecting the flow of the page.

translate(x,y)

transform: translate(45px, -45px);

  1. Is not natively supported on IE9 and other older browsers unless you do the following.

-webkit-transform: translate(45px,-45px); /* iOS8, Android

↵4.4.3, BB10 */

-ms-transform: translate(45px,-45px); /* IE9 only */

transform: translate(45px,-45px);

  1. You can move things on the X and Y axis separately

transform: translateX(45px);

transform: translateY(-30px);

  1. Scaling
    1. Scales an element by the defined factors horizontally then vertically.

scale(x,y)

transform: scale(1.5, 0.25);

  1. Rotation
    1. Rotates an element around the point of origin by a specific angle value.

rotate()

.ad-ad2 h1:hover span {

    color: #484848;

    transform: rotate(10deg) translateX(40px) scale(1.5);

}

  1. Skew
    1. Skew along the x and y axes

skew(x,y)

transform: skew(15deg, 4deg);

  1. Changing the origin of the Transform
    1. Control the origin from which your transformation is applied.

transform-origin

transform-origin: 0 0;

  1. Support for IE8 and Earlier

.translate {

    position: relative;

    top: 200px;

    left: 200px;

}

Transitions

  1. CSS transitions require less client-side processing, so they appear smoother.
    1. Transition-property
    2. Transition-duration
    3. Transition-timing-function
    4. Transition-delay
  1. Webkit is still needed for older browsers

transition-property

.ad-ad2 h1 span {

    transition-property: transform;

}

  1. Older browsers

.ad-ad2 h1 span {

    -webkit-transition-property: -webkit-transform;

    transition-property: transform;

}

  1. .foo {

    transition-property: transform, color;

}

  1. Transition-duration sets how long the transition will take in seconds(s) or milliseconds(ms).

transition-duration: 0.2s;

  1. Transition-timing-function controls the pace of the transition in more detail.

transition-timing-function: ease-out;

  1. Transition-delay introduce a delay prior to a transition happens.

-webkit-transition-delay: 50ms;

transition-delay: 50ms;

  1. Transition shorthand property
    1. Long way

.ad-ad2 h1 span {

    transition-duration: 0.2s; 

    transition-timing-function: ease-out;

    transition-delay: 50ms;

}

  1. Short hand

}

  1. Multiple transitions can be called in one call
    1. Example A

  1. Example B

  1. Example D - effects all properties at the same time.
